# 科技感简历

这是一个具有科技感的个人简历项目，使用HTML、CSS和JavaScript构建。

## 项目结构

```
├── index.html           # 主HTML文件
├── css/                 # 样式文件
│   └── style.css        # 主样式文件
├── js/                  # JavaScript文件
│   └── main.js          # 主脚本文件
├── images/              # 图片资源
│   ├── dylan-logo.svg
│   └── dylan-logo2.svg
└── README.md            # 项目说明
```

## 功能特点

- 响应式设计，适配不同设备
- 科技感UI设计
- 交互动效
- 简洁明了的内容展示

## 如何使用

直接在浏览器中打开`index.html`文件即可查看简历。
